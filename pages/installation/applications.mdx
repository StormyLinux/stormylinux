# Required Applications

## Installing required applications

Let's install all required applications on Debian.

| Instructions/Commands         |           Application                         |
|:---|:---|
| `sudo apt install terminator`       | Terminator (Multiple terminals in on window)  |
| `sudo apt install code`             | VSCode (IDE)                                  |
| `sudo apt install ranger`           | Ranger (Terminal based file explorer)         |
| `sudo apt install krusader`         | Krusader (Advanced file explorer)             |
| `sudo apt install rofi`             | Rofi (Customizable Menu)                      |
| `sudo apt install ueberzug`         | Ueberzug (Preview Images in Ranger)           |
| `sudo apt install brightnessctl`    | BrightnessCTL (Control Screen Brightness)     |
| `sudo apt install vim`              | VIM (Programming Text Editor)                 |
| `sudo apt install neovim`           | NVIM (Customizable Programming Text Editor)   |
| `sudo apt install bookworm`         | Bookworm (Ebook and PDF reader)               |
| `sudo apt install rhythmbox`        | Rhythmbox (Music Player)                      |
| `sudo apt install picom`            | Bookworm (Ebook and PDF reader)               |
| `sudo apt install cava`             | CAVA (Monitor Music Sound Levels)             |
| `sudo apt install thunderbird`      | Thunderbird (Ebook and PDF reader)            |
| `sudo apt install gnome-screenshot` | Thunderbird (Ebook and PDF reader)            |
| `sudo apt install subl` or download [here](https://www.sublimetext.com/download)                     | Sublime-Text (Text-Editor)                    |
| [GitHub Desktop Installation](/install-github-desktop)                                               | GitHub Desktop                                |
| [Lunacy Installation](https://icons8.com/lunacy)                                                     | Lunacy (Web Design tool & Figma Alternative)  |
| [Obsidian Installation](https://obsidian.md/)                                                        | Obsidian (Notes as MD files)                  |
| [Thorium Installation](https://github.com/Alex313031/Thorium/releases)                               | Thorium (Super Secure Browser)                |
| [Brave Installation](https://brave.com/linux/)                                                       | Brave (Secure Browser)                        |
| [Alacritty Installation](https://github.com/alacritty/alacritty/blob/master/INSTALL.md)              | Alacritty (Customizable Terminal)             |
| [Kitty Installation](https://sw.kovidgoyal.net/kitty/binary/)                                        | Kitty (GPU Accelerated Terminal)              |
| [Spotify Installation](https://www.spotify.com/de-en/download/linux/)                                | Spotify (Music)                               |
| [Stacer Installation](https://oguzhaninan.github.io/Stacer-Web/)                                     | Stacer (Control System Processes)             |
| [QBittorrent Installation](https://www.qbittorrent.org/download)                                     | QBittorrent (Control System Processes)        |
| [Anaconda Navigator  Installation](https://docs.anaconda.com/free/navigator/index.html)              | Anaconda Navigator (Data Science)             |
