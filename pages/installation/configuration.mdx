# Configuration

## The Config File

Configuring Stormy Linux is a breeze, and to get started, open up a terminal then navigate to the directory of your default configuration folder of Qtile, navigate into the `.config` directory, located at `~/.config/qtile/config.py`, then open the `config.py` Python file using a text editor of your choice (vim, nano, xed, etc).

> A basic understanding of Python is essential to configuring Stormy Linux to your needs.

The Stormy Linux configuration is aimed at making it easy enough for anyone to dive in and customize it to their needs.
