## Brightness

These shortcuts rely on the `systemctl` brightness control package.

| Command                           | Description                        |
|:---|:---|
| `mod` + `shift` + `a`             | Brightness 10%                     |
| `mod` + `shift` + `b`             | Brightness 20%                     |
| `mod` + `shift` + `c`             | Brightness 30%                     |
| `mod` + `shift` + `d`             | Brightness 40%                     |
| `mod` + `shift` + `e`             | Brightness 50%                     |
| `mod` + `shift` + `f`             | Brightness 60%                     |
| `mod` + `shift` + `g`             | Brightness 70%                     |
| `mod` + `shift` + `h`             | Brightness 80%                     |
| `mod` + `shift` + `i`             | Brightness 90%                     |
| `mod` + `shift` + `j`             | Brightness 100%                    |

## Layout

| Command                           | Description                        |
|:---|:---|
| `mod` + `z`                       | Makes a window fullscreen          |
| `mod` + `left`                    | Switch to window on the left       |
| `mod` + `right`                   | Switch to window on the right      |
| `mod` + `up`                      | Switch to window above             |
| `mod` + `down`                    | Switch to window below             |

## Switch Focus

| Command                           | Description                        |
|:---|:---|
| `mod` + `i`                       | Switch to screen 0                 |
| `mod` + `o`                       | Switch to screen 1                 |

## Focus Monitor

| Command                           | Description                        |
|:---|:---|
| `mod` + `,`                       | Previous Screen                    |
| `mod` + `.`                       | Next Screen                        |
