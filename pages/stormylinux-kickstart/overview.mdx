# Kickstart Edition Overview

## Stormy Linux Kickstart

The Kickstart Edition of Stormy Linux features a minimal look and feel with a single configuration file where you can customize and change the look and feel of your system. This configuration file includes documentation to get you started on understanding how to quickly configure your system.

This Kickstart Edition was made for beginners that are just getting started with window managers and would like to have a user-friendly and extensible work environment that is highly customizable and easy to use.

> Some basic knowledge of the Python programming language will be an advantage and will help make the configuration process much easier.

The Kickstart Edition leverages the Qtile window manager to provide a comfortable, minimal and customizable work environment for maximum productivity.
