# Keybindings

## Workspace Selection

### Command and Description

We will be referencing the `windows key` by using `mod`, and the `Alt` key will be referenced as `mod1`.

`**Windows Key**` = `**mod**`
`**Alt Key**` = `**mod1**`

| Command                           | Description                        |
|:---|:---|
| `mod` + `1`                       | Move to workspace 1                |
| `mod` + `2`                       | Move to workspace 2                |
| `mod` + `3`                       | Move to workspace 3                |
| `mod` + `4`                       | Move to workspace 4                |
| `mod` + `5`                       | Move to workspace 5                |
| `mod` + `6`                       | Move to workspace 6                |
| `mod` + `7`                       | Move to workspace 7                |
| `mod` + `8`                       | Move to workspace 8                |
| `mod` + `9`                       | Move to workspace 9                |
| `mod` + `10`                      | Move to workspace 10               |
| `mod` + `11`                      | Move to workspace 11               |
| `mod` + `12`                      | Move to workspace 12               |
| `mod` + `F1`                      | Move to workspace 13               |
| `mod` + `F2`                      | Move to workspace 14               |
| `mod` + `F3`                      | Move to workspace 15               |
| `mod` + `F4`                      | Move to workspace 16               |

## Brightness

These shortcuts rely on the `systemctl` brightness control package.

| Command                           | Description                        |
|:---|:---|
| `mod` + `mod1` + `1`             | Brightness 10%                     |
| `mod` + `mod1` + `2`             | Brightness 20%                     |

## Layout

| Command                           | Description                        |
|:---|:---|
| `mod` + `z`                       | Makes a window fullscreen          |
| `mod` + `left`                    | Switch to window on the left       |
| `mod` + `right`                   | Switch to window on the right      |
| `mod` + `up`                      | Switch to window above             |
| `mod` + `down`                    | Switch to window below             |
| `mod` + `shift` + `space`         | Toggle floating                    |
| `mod` + `Tab`                     | Next group                         |
| `mod` + `shift` + `Tab`           | Previous group                     |


## Window Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `q`                       | Close current window               |
| `mod` + `control` + `space`       | Toggle fullscreen                  |

## Qtile System Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `control` + `r`           | Restart Qtile                      |
| `mod` + `control` + `q`           | Quit Qtile(Logout)                 |

## Switch Focus

| Command                           | Description                        |
|:---|:---|
| `mod` + `i`                       | Switch to screen 0                 |
| `mod` + `o`                       | Switch to screen 1                 |

## Focus Monitor

| Command                           | Description                        |
|:---|:---|
| `mod` + `,`                       | Previous Screen                    |
| `mod` + `.`                       | Next Screen                        |


## System Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `delete`                  | Power Off System                   |
| `mod` + `mod1` + `delete`         | Reboot System                      |
| `mod` + `control` + `q`           | Log Out System                     |
