# Keybindings

A list of predefined keybindings for Stormy Linux Developer Edition.

## Workspace Selection

### Command and Description

We will be referring to the `Windows Key` as `Super` and the `Enter Key` as `Return`.

`Windows Key` = `Super`

`Enter Key` = `Return`

<br/>

## Window Focus

| Command                             | Description                        |
|:---|:---|
| `Super` + `Up`                      | Switch to window above             |
| `Super` + `Down`                    | Switch to window below             |
| `Super` + `Left`                    | Switch to left window              |
| `Super` + `Right`                   | Switch to right window             |

## Layout

| `Super` + `Shift` + `Space`         | Toggle floating                    |
| `Super` + `Tab`                     | Next group                         |
| `Super` + `Shift` + `Tab`           | Previous group                     |

## Window Actions

| Command                             | Description                        |
|:---|:---|
| `Super` + `q`                       | Close current window               |
| `Super` + `Ctrl` + `Space`          | Toggle fullscreen                  |

## Qtile System Actions

| Command                             | Description                        |
|:---|:---|
| `Super` + `Ctrl` + `r`              | Restart Qtile                      |
| `Super` + `Ctrl` + `q`              | Quit Qtile(Logout)                 |

## Switch Focus

| Command                             | Description                        |
|:---|:---|
| `Super` + `Alt` + `i`               | Switch to screen 0                 |
| `Super` + `Alt` + `o`               | Switch to screen 1                 |

## Focus Monitor

| Command                             | Description                        |
|:---|:---|
| `Super` + `,`                       | Previous Screen                    |
| `Super` + `.`                       | Next Screen                        |


## System Actions

| Command                             | Description                        |
|:---|:---|
| `Super` + `delete`                  | Power Off System                   |
| `Super` + `Alt` + `delete`          | Reboot System                      |
| `Super` + `Ctrl` + `q`              | Log Out System                     |
