# Keybindings

## Workspace Selection

### Command and Description

We will be referring to the `windows key` as `mod`.

`Windows Key` = `mod`

| Command                           | Description                        |
|:---|:---|
| `mod` + `1`                       | Move to workspace 1                |
| `mod` + `2`                       | Move to workspace 2                |

## Brightness

These shortcuts rely on the `systemctl` brightness Ctrl package.

| Command                           | Description                        |
|:---|:---|
| `mod` + `Alt` + `1`             | Brightness 10%                     |

## Layout

| Command                           | Description                        |
|:---|:---|
| `mod` + `left`                    | Switch to left window              |
| `mod` + `right`                   | Switch to right window             |
| `mod` + `up`                      | Switch to window above             |
| `mod` + `down`                    | Switch to window below             |
| `mod` + `Shift` + `space`         | Toggle floating                    |
| `mod` + `Tab`                     | Next group                         |
| `mod` + `Shift` + `Tab`           | Previous group                     |


## Window Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `q`                       | Close current window               |
| `mod` + `Ctrl` + `space`       | Toggle fullscreen                  |

## Qtile System Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `Ctrl` + `r`           | Restart Qtile                      |
| `mod` + `Ctrl` + `q`           | Quit Qtile(Logout)                 |

## Switch Focus

| Command                           | Description                        |
|:---|:---|
| `mod` + `i`                       | Switch to screen 0                 |
| `mod` + `o`                       | Switch to screen 1                 |

## Focus Monitor

| Command                           | Description                        |
|:---|:---|
| `mod` + `,`                       | Previous Screen                    |
| `mod` + `.`                       | Next Screen                        |


## System Actions

| Command                           | Description                        |
|:---|:---|
| `mod` + `delete`                  | Power Off System                   |
| `mod` + `Alt` + `delete`         | Reboot System                      |
| `mod` + `Ctrl` + `q`           | Log Out System                     |
